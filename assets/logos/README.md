# Mythos logo archive

Every logo iteration designed for Mythos, kept for reuse (releases, social
media, alternate icons, future rebrands). All are self-contained SVGs — no
external fonts or images.

**The current official logo is [`../logo.svg`](../logo.svg)** (same as
`07-icon-cosmic-knight.svg`).

| File | Style | Notes |
| --- | --- | --- |
| `01-circle-gold-royal.svg` | Circular badge | First concept: gold knight on royal blue |
| `02-circle-amethyst-knight.svg` | Circular badge | Violet knight, amethyst palette chosen |
| `03-circle-pegasus-wings.svg` | Circular badge | Added golden Pegasus wings, moon, glowing eye |
| `04-banner-wordmark.svg` | Wide banner (800×320) | Gold-framed banner with a MYTHOS wordmark |
| `05-icon-square-winged.svg` | Rounded-square icon | Winged knight in a gold-framed square |
| `06-icon-ivory-knight.svg` | App icon (Stockfish-style) | Ivory knight tilted over a 2×2 board with neural constellations |
| `07-icon-cosmic-knight.svg` | App icon — **current** | The cosmic knight: star-filled body, gold outline, wings, moon, laurel wreath |
| `08-color-options.svg` | Exploration sheet | The four candidate color schemes (amethyst / emerald / teal / crimson) |

The knight silhouette is the **cburnett** chess set (the Lichess default piece
set), GPL — consistent with the pieces used in the browser UI.
